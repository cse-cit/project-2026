# project-2026

A Novel Methodology Of An Ai Based Smart Attendance System

final year project repo for cse'22 batch
The SnapClass project successfully developed an AI-based smart attendance management system that automates attendance recording using face and voice recognition technologies. The system was designed to overcome the limitations of traditional attendance methods, such as manual effort, time consumption, and proxy attendance.

By integrating facial recognition, voice biometrics, QR code-based enrollment, and cloud-based data storage, the system provides a secure, efficient, and user-friendly attendance solution. The face recognition module accurately identifies students from classroom photographs, while the voice recognition module serves as an alternative attendance verification method. The use of Supabase cloud storage ensures secure management and easy access to attendance records.

The testing results demonstrated that the system achieved high accuracy and reliable performance under real classroom conditions. Automated attendance processing significantly reduced the time required for attendance recording and improved overall efficiency. The separate teacher and student portals further enhanced usability and attendance management.

Overall, SnapClass proves that Artificial Intelligence can be effectively utilized to modernize attendance systems. The project provides a cost-effective, scalable, and reliable solution for educational institutions while improving accuracy, security, and convenience in attendance management.
